# 🧠 PsycheMap Blog

Built with [Hugo](https://gohugo.io/) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod) and deployed to GitHub Pages.

## Local Development

```bash
hugo server --buildDrafts
```

## Creating a New Post

```bash
hugo new content posts/my-new-post.md
```

## Deploying

Push to `main` — GitHub Actions builds and deploys automatically.
